Detailed Description

Ponto de entrada do executável TCGGUI.

Inicializa o banco de dados (via Singleton NexusDbManager, da DLL TCGCore), exibe o LoginDialog e, de acordo com o perfil autenticado, abre a DevPortalWindow (ADMIN) ou a StoreDashWindow (LOJISTA).
