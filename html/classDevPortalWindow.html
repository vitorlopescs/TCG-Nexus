Detailed Description

Janela principal do perfil ADMIN.

Concentra dois casos de uso da Sprint 1: sincronizar a base de dados de cartas a partir de um arquivo JSON (Requisito 1) e cadastrar novos usuários com seu respectivo perfil de acesso (Requisitos 4 e 5).
